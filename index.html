HTML Tag 의 종류

  • br = 줄바꿈
  • p = 단락으로 줄바꿈
  • i = 글자 기울이기
  • b = 글자 굵게 하기
  • strong = 글자 강조하기
  • h1~h6 제목글
  • href(hypertext reference) = 인터넷 링크, target:_self, _black, title
  • a(anchor) = 링크의 시작
  • img = 그림 넣기, src width height
  • div = 레이아웃, style
  • span = 레이아웃, div 와 다른점은 줄바꿈이 안되는거다
  • ul = unordered list, ol = ordered list
